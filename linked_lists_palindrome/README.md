# INSERT IN SORTED LINKED LIST

## Purpose
Checks if a singly linked list forms a palindrome with its values

- Method Type: **`Iterative`**
- Libraries: **`stdio.h`**, **`stdlib.h`**

## Compilation
```bash
holbie@VM-Algos:~/insert_in_sorted_linked_list$ gcc -Wall -Werror -Wextra -pedantic 0-main.c linked_lists.c 0-is_palindrome.c -o palindrome
```

## Execution Process (Output)
```bash
holbie@VM-Algos:~/linked_lists_palindrome$ ./palindrome
1
```
